0.  How many different colors does each format support ?
    BMP: up to 16 million (typically 8 bit file, there is support for up to 24 bits)
    GIF: up to 256 (8 bit color palette)
    JPEG: up to 16.7 million (24 bit color palette)
    PNG: up to 16 million (8-bit color but also extentable to 24-bits)
    
1.  Which of the formats supports animation ?
    GIF

2.  What's the difference between lossy and lossless compression ?
    Lossy compression make a copy of the perfect photo by preserving
    representation, yet it isn't perfect copy of the original. 
    However, lossless one both preserve represenatation and create 
    a perfect copy with smaller size, still its size may be larger than 
    those in created with lossy compression.
    
3.  Which of these formats is lossy-compressed ?
    JPEG
    
4.  What happens when a file is deleted from a FAT file system? FAT stands for
	File Allocation Table; when a file is deleted, the information in the
	address table is erased, but the file itself remains on the hard drive.

5.  What can someone like you do to ensure (with high probability) that files you delete cannot be recovered? 
	Delete files using a satitation tools. That replaces the section on disc with all zeroes.
